Forky App is a web application project developed as part of the Udemy course The Complete JavaScript Course 2025: From Zero to Expert! by Jonas Schmedtmann.

This project serves as a hands-on learning exercise to strengthen my understanding of modern JavaScript. The primary focus is on asynchronous JavaScript and the behavior of the this keyword in different execution contexts. Through this project, I aim to build a strong foundation in JavaScript that will support my transition into Node.js and backend development in the future.

## Key Learnings

- Asynchronous JavaScript (Promises, async/await, and the event loop)
- Understanding and applying the this keyword in various contexts
- Consuming and handling data from external APIs
- Managing asynchronous application state
- Applying the MVC (Model-View-Controller) architectural pattern
